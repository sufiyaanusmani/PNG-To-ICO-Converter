
# PNG To ICO Converter

This program will convert .png image to .ico

## Run Locally

Clone the project

```bash
  git clone git@github.com:sufiyaanusmani/PNG-To-ICO-Converter.git
```

Go to the project directory

```bash
  cd PNG-To-ICO-Converter
```

Start the program

```bash
  python app.py
```
